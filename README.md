## 🌌 The Platform

**Nexus AI** is a state-of-the-art enterprise intelligence workspace built for high-performance teams. It brings together multi-model switching, document parsing (RAG), visual agent workflows, and natural language data analytics into a single, cohesive, monochromatic interface.

---

## ✨ Key Features

| Feature | Description |
|---|---|
| 🤖 **Multi-Model Hub** | Seamlessly route tasks between Claude, GPT-4, and Gemini. |
| 📊 **Data Analytics** | Natural language to SQL via DuckDB. Export to PDF, DOCX, or PPTX. |
| 🧠 **Document Intelligence** | Upload resources, generate summaries, and populate vector storage. |
| ⚡ **Visual Workflows** | Wire document ingestion, OCR, translation, and alerts visually. |
| 🛡️ **Admin Panel** | Role-based access control, security audit logs, and server allocations. |

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Vinisha-10/Nexas_ai_frontend.git
   cd Nexas_ai_frontend
   ```

2. **Install dependencies**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy

Inspired by the clean aesthetics of Vercel and Linear, Nexus AI features a **Monochromatic Black** design system (`#080808` base). It prioritizes typography (`Syne` for display, `Inter` for body), readability, and absolute minimal use of color for functional alerts only.
